# Mylibint Test

This is a basic example for the proper use of my library in a simple TS proyect.

## Usage
clone the repo
install dependencies 
```javascript 
npm i
```
run the tsc of index.ts
```javascript 
tsc index  
```
and finally run the index.js
```javascript 
node index.js
```
